# LibraryProject - Django Development Environment Setup

## Project Overview
LibraryProject is a Django-based application created as part of the ALX Django Learning Lab. This project serves as an introduction to Django development, covering project setup, configuration, and basic structure exploration.

## Prerequisites
- Python 3.8 or higher installed on your system
- pip (Python package installer)
- Basic command line knowledge

## Installation Steps

### 1. Install Django
```bash
pip install django
```

Verify Django installation:
```bash
django-admin --version
```

### 2. Navigate to Project Directory
```bash
cd LibraryProject
```

### 3. Run the Development Server
```bash
python manage.py runserver
```
